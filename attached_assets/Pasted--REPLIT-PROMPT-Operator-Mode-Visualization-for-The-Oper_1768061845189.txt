## INTERACTION FLOW SUMMARY

1. User sees OPERATOR MODE button on home/nav
2. First time: Goes through setup flow (5 screens)
3. After setup: Taps button → sees activation confirmation
4. Taps "GO TO WAR" → Screen transforms to active state
5. Timer counts down, shows tasks, grayscale effect
6. If exit early → warning with consequences
7. Timer ends → completion celebration screen
8. Returns to normal app
